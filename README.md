# Secure-Personal-Finance-Tracker
A full-stack web application that helps users track, organize, and analyze their personal expenses. The app allows users to securely add, update, and delete financial transactions while visualizing spending patterns through interactive charts and dashboards.

Features:

Secure authentication (Node.js + Express)

Add, edit, delete & categorize expenses

Detailed transaction history with filters

Interactive spending analytics (Chart.js)

Responsive UI built with HTML, CSS, and JavaScript

PostgreSQL relational database for structured financial data

RESTful API architecture for clean separation between frontend & backend

Tech Stack:

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: PostgreSQL

Visualization: Chart.js

Tools: Git, VS Code, Postman

Project Goals:

To provide users with an easy-to-use platform that enables better financial awareness by visualizing income and expense trends over time.

Future Enhancements:

Monthly budgeting module

AI-based spending insights

Export reports (PDF/CSV)

Login using OAuth (Google/Microsoft)
